# Projeto Shalom
Landing page institucional desenvolvida para o Projeto Shalom, organização social atuante em Betim, Minas Gerais. O sistema tem como objetivo centralizar a divulgação de atividades socioeducativas e facilitar a captação de recursos e voluntariado.

1. Visão Geral
O projeto consiste em uma interface web moderna focada em usabilidade e performance, apresentando as frentes de atuação da instituição: esporte, reforço escolar e assistência social. O desenvolvimento prioriza a acessibilidade e a responsividade, garantindo a entrega do conteúdo em múltiplos dispositivos.

2. Tecnologias Empregadas
Linguagem de Marcação: HTML5 com foco em semântica.

Estilização: CSS3 utilizando metodologias de Flexbox e Grid Layout.

Variáveis de Estilo: Implementação de tokens de design para cores e tipografia, facilitando a manutenção.

Tipografia: Utilização da família DM Sans para otimização da legibilidade.

3. Características Técnicas
Layout Adaptativo: Interface totalmente responsiva para desktop e dispositivos móveis.

Componentização: Estrutura de footer e navegação otimizada para escalabilidade.

Integração Social: Pontos de contato e redes sociais estrategicamente posicionados para conversão.
